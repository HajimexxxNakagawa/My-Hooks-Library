## This Next.js project is compatible with ...

- [x] TypeScript
- [x] CSS Modules
- [x] Prettier & ESLint
- [x] lint-staged & husky
- [x] Storybook
- [ ] PWA
- [ ] State Management

## Command

```
# dev
yarn dev

# build
yarn build

# start
yarn start
```

## Storybook

```
yarn storybook

# build
yarn build-storybook
```

## Lint

```
yarn lint

# auto fix
yarn lint --fix

# lint for git stage
yarn lint-staged
```
